# Bicameral Mind

Agentic system with bicameral architecture, procedural memory, and MCP integration.

## Quick Start

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   python main.py

3. (Optional) Run tests:
   python -m pytest

## Notes

- Default model: qwen3:14b (Ollama).
- Data directories are created on first run.
